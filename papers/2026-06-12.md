# Daily Papers — 2026-06-12

> Topics: LLM for Code, LLM for Software Engineering

## Daily Highlight

Today's papers reveal a strong focus on **agentic systems for complex reasoning and scientific discovery**, with multiple works addressing how LLM-based agents can handle dynamic environments, multi-modal interactions, and tool orchestration more effectively. Key trends include moving beyond step-wise tool calls toward more sophisticated action interfaces, improving knowledge representation in agent reasoning, and scaling agent-based approaches to autonomous scientific discovery. Notably, several papers tackle compositional reasoning failures and retrieval-augmented strategies, suggesting the community is actively addressing fundamental limitations in LLM reasoning.

## Papers by Theme

### Agents & Agent Architecture
- **[EvoArena: Tracking Memory Evolution for Robust LLM Agents in Dynamic Environments](https://arxiv.org/abs/2606.13681v1)** — Introduces a benchmark suite for evaluating LLM agents in dynamic, non-static environments where task conditions continually evolve.
- **[InterleaveThinker: Reinforcing Agentic Interleaved Generation](https://arxiv.org/abs/2606.13679v1)** — Enables LLM-based agents to generate interleaved text-image sequences for applications in visual narratives and embodied manipulation.
- **[HyperTool: Beyond Step-Wise Tool Calls for Tool-Augmented Agents](https://arxiv.org/abs/2606.13663v1)** — Proposes hierarchical tool workflows that reduce execution-granularity mismatch and context overhead by bundling atomic tool calls into higher-level operations.
- **[SpatialClaw: Rethinking Action Interface for Agentic Spatial Reasoning](https://arxiv.org/abs/2606.13673v1)** — Redesigns action interfaces for vision-language agents to improve spatial reasoning and 3D object understanding.

### Scientific Discovery & Knowledge Agents
- **[EurekAgent: Agent Environment Engineering is All You Need For Autonomous Scientific Discovery](https://arxiv.org/abs/2606.13662v1)** — Argues that the bottleneck for LLM-based autonomous scientific discovery has shifted from model capability to environment engineering.
- **[Agents-K1: Towards Agent-native Knowledge Orchestration](https://arxiv.org/abs/2606.13669v1)** — Introduces structured knowledge orchestration for research agents, moving beyond flat citation handling to capture entities, claims, evidence, and method lineages.

### Reasoning & Compositionality
- **[Operadic consistency: a label-free signal for compositional reasoning failures in LLMs](https://arxiv.org/abs/2606.13649v1)** — Proposes operadic theory as a diagnostic framework for detecting compositional reasoning failures in LLMs without ground-truth labels.
- **[Learning to Reason by Analogy via Retrieval-Augmented Reinforcement Fine-Tuning](https://arxiv.org/abs/2606.13680v1)** — Develops retrieval-augmented methods that surface analogous reasoning strategies rather than semantically similar problems for complex reasoning tasks.

### Data & Model Improvement
- **[Influcoder: Distilling Decoders' Gradient Influence Rankings into an Encoder for Data Attribution](https://arxiv.org/abs/2606.13668v1)** — Proposes efficient data attribution by distilling gradient influence rankings into an encoder to identify high-quality training samples.
- **[Automated reproducibility assessments in the social and behavioral sciences using large language models](https://arxiv.org/abs/2606.13670v1)** — Demonstrates that LLMs can automate reproducibility assessment of published studies at scale, reducing manual review overhead.

### World Models & Robotics
- **[RepWAM: World Action Modeling with Representation Visual-Action Tokenizers](https://arxiv.org/abs/2606.13674v1)** — Introduces representation-centric world action models using visual-action tokenizers that prioritize instruction-following dynamics over pixel reconstruction.
- **[WEAVER: Better, Faster, Longer: An Effective World Model for Robotic Manipulation](https://arxiv.org/abs/2606.13672v1)** — Presents a world model optimized jointly for fidelity, efficiency, and horizon length to enable policy evaluation, improvement, and planning in robotics.

### Out of Scope (Non-LLM/Code Papers)

- **[Mana: Dexterous Manipulation of Articulated Tools](https://arxiv.org/abs/2606.13677v1)** — Robotic manipulation without LLM agents.
- **[Improving Robotic Generalist Policies via Flow Reversal Steering](https://arxiv.org/abs/2606.13675v1)** — Flow-based robotic policy improvement unrelated to LLMs.
- **[Modality Forcing for Scalable Spatial Generation](https://arxiv.org/abs/2606.13676v1)** — Vision model adaptation for depth prediction, not LLM-focused.
