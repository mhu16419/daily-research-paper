# Daily Papers — 2026-06-16

> Topics: LLM for Code, LLM for Software Engineering

## Daily Highlight

Today's papers show a strong focus on **LLM agents and reasoning optimization**, with multiple works tackling context management, reinforcement learning for long-horizon tasks, and evidence-based decision-making in complex domains like scientific research and meta-analysis. Key advances include cache-efficient context handling, rubric-supervised RL for research agents, and better understanding of how LLMs internally represent value and confidence during reasoning.

## Papers by Theme

### LLM Agents & Agentic Systems
- **[Context-Aware RL for Agentic and Multimodal LLMs](https://arxiv.org/abs/2606.17053v1)** — Proposes ContextRL to improve LLM agent performance on long-horizon reasoning and multimodal tasks by identifying decisive evidence within complex contexts through indirect auxiliary objectives.
- **[Benchmarking LLM Agents on Meta-Analysis Articles from Nature Portfolio](https://arxiv.org/abs/2606.17041v1)** — Introduces MetaSyn, a benchmark dataset of 442 meta-analysis tasks with ground truth across retrieval-screening-synthesis pipelines for evaluating systematic scientific reasoning in LLM agents.
- **[DEEPRUBRIC: Evidence-Tree Rubric Supervision for Efficient Reinforcement Learning of Deep Research Agents](https://arxiv.org/abs/2606.17029v1)** — Uses evidence-tree rubric-based reward supervision to improve RL efficiency for research agents that synthesize long-form reports by optimizing against checkable quality criteria.

### Context Management & Efficiency
- **[TokenPilot: Cache-Efficient Context Management for LLM Agents](https://arxiv.org/abs/2606.17016v1)** — Addresses context accumulation costs in long-horizon LLM agent sessions by balancing text sparsity and prompt cache continuity to avoid cache invalidation from sequence mutations.
- **[KVEraser: Learning to Steer KV Cache for Efficient Localized Context Erasing](https://arxiv.org/abs/2606.17034v1)** — Enables post-hoc context erasing over the KV cache for long-context LLM applications by learning to steer cached states when stale facts or harmful injections need removal.

### Reasoning & Value Learning
- **[The Value Axis: Language Models Encode Whether They're on the Right Track](https://arxiv.org/abs/2606.17056v1)** — Demonstrates that language models internally encode a "value axis" that tracks the likelihood their current strategy will achieve goals, distinguishing high from low confidence trajectories.
- **[ExpRL: Exploratory RL for LLM Mid-Training](https://arxiv.org/abs/2606.17024v1)** — Improves sparse-reward RL for LLM reasoning by combining mid-training on curated reasoning traces with exploratory RL to overcome coverage limitations in base models.

### Evaluation & Benchmarking
- **[Bayesian Inference and Decision Audits for Public Archives of Frontier AI Evaluations](https://arxiv.org/abs/2606.17005v1)** — Provides longitudinal statistical analysis of selective public AI evaluation time series (LiveBench, Open LLM Leaderboard v2) to account for reporting rules, benchmark revisions, and missingness in leaderboard interpretations.
