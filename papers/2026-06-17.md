## Papers by Theme

### LLM-Based Agents & Software Engineering
- **[ReproRepo: Scaling Reproducibility Audits with GitHub Repository Issues](https://arxiv.org/abs/2606.18237v1)** — Introduces a scalable framework leveraging LLM agents to evaluate and assist with research reproducibility across GitHub repositories, reducing manual curation overhead.

- **[Seeing Is Not Screening: Multimodal Hidden Instruction Attacks on Agent Skill Scanners](https://arxiv.org/abs/2606.18198v1)** — Reveals security vulnerabilities in LLM-based agent skill scanners that rely primarily on text and code analysis, missing visually-conveyed malicious instructions.

### Model Architecture & Scaling
- **[Variable-Width Transformers](https://arxiv.org/abs/2606.18246v1)** — Proposes nonuniform capacity allocation across transformer layers rather than fixed width, improving efficiency by allowing different layers to specialize in distinct computational roles.

- **[Fixed-Point Reasoners: Stable and Adaptive Deep Looped Transformers](https://arxiv.org/abs/2606.18206v1)** — Addresses signal propagation issues in looped transformer architectures designed for step-by-step reasoning by introducing stability mechanisms analogous to fixed-point iterations.

### Model Training & Optimization
- **[Learning from the Self-future: On-policy Self-distillation for dLLMs](https://arxiv.org/abs/2606.18195v1)** — Extends on-policy self-distillation to diffusion language models, adapting techniques from autoregressive models to the non-sequential generation paradigm.

- **[Zone of Proximal Policy Optimization: Teacher in Prompts, Not Gradients](https://arxiv.org/abs/2606.18216v1)** — Proposes using teacher models to improve student LLM generalization through prompt-based guidance rather than logit imitation, addressing brittleness in knowledge distillation.

### Evaluation & Benchmarking
- **[RubricsTree: Scalable and Evolving Open-Ended Evaluation of Personal Health Agents across Health Memory and Medical Skills](https://arxiv.org/abs/2606.18203v1)** — Develops an evaluation framework balancing physician reliability with LLM-as-a-judge scalability for assessing health agent competence, applicable beyond healthcare domains.

- **[EBench: Elemental Diagnosis of Generalist Mobile Manipulation Policies](https://arxiv.org/abs/2606.18239v1)** — Provides a diagnostic benchmark for evaluating generalist manipulation models across multiple capability and generalization dimensions beyond single success metrics.
