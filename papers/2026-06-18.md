# Daily Papers — 2026-06-18

> Topics: LLM for Code, LLM for Software Engineering

## Daily Highlight

Today's collection shows a strong emphasis on **LLM-based agents for software engineering**, with multiple papers advancing multi-agent systems, test-time scaling, and design patterns for complex SE tasks. Beyond agents, there is notable work on **LLM-based security analysis** (malware detection, code analysis) and **evaluator validation**, addressing the critical challenge of ensuring LLMs reliably assess code artifacts. Additionally, emerging techniques like diffusion-based theorem proving and program synthesis for interpretability suggest the field is expanding beyond traditional auto-regressive generation paradigms.

## Papers by Theme

### Agents & Multi-Agent Systems
- **[Trae Agent: An LLM-based Agent for Software Engineering with Test-time Scaling](https://www.semanticscholar.org/paper/15a9b53dc8313eb0542dac00e1ea6e3dad2bbf51)** — Introduces test-time scaling techniques to enhance LLM-based software issue resolution through ensemble reasoning.
- **[LLM-Based Multi-Agent Systems for Software Engineering: Literature Review, Vision, and the Road Ahead](https://www.semanticscholar.org/paper/027d9a3371c13ea567bca75e675db295022a300b)** — Surveys the integration of LLMs into autonomous multi-agent systems for addressing complex software engineering challenges.
- **[From LLMs to LLM-based Agents for Software Engineering: A Survey of Current, Challenges and Future](https://www.semanticscholar.org/paper/987c5ab507b2fce644a9b73a8f8338c80cba944d)** — Comprehensive survey examining LLM capabilities, limitations, and the transition to LLM-based agent architectures in SE.
- **[Nemotron-CORTEXA: Enhancing LLM Agents for Software Engineering Tasks via Improved Localization and Solution Diversity](https://www.semanticscholar.org/paper/34d96fe3d9c45aad9021cc20557a054a35df935f)** — Improves LLM agent performance through better fault localization and diverse solution generation.
- **[Designing LLM-based Multi-Agent Systems for Software Engineering Tasks: Quality Attributes, Design Patterns and Rationale](https://www.semanticscholar.org/paper/4fe99230d170b6c17570ec1cb4b10c3e99aef56e)** — Provides design patterns and quality attributes for constructing effective LLM-based multi-agent SE systems.
- **[Data Intelligence Agents: Interpreting, Modeling, and Querying Enterprise Data via Autonomous Coding Agents](https://arxiv.org/abs/2606.19319v1)** — Presents a three-agent system that automates enterprise data integration workflows using autonomous coding agents.

### Security & Code Analysis
- **[Large Language Model (LLM) for Software Security: Code Analysis, Malware Analysis, Reverse Engineering](https://www.semanticscholar.org/paper/605a8b0d425a7cf6a0e97eafa681e656081d958e)** — Reviews LLM applications in cybersecurity, including malware detection, code analysis, and vulnerability identification.

### LLM Evaluation & Validation
- **[Automated Validation of LLM-based Evaluators for Software Engineering Artifacts](https://www.semanticscholar.org/paper/01b881d5a3471d4c0270b07c77604b9ebdf5fc95)** — Introduces REFINE framework to automate validation of LLM evaluators for code quality assessment without costly human annotation.

### Formal Reasoning & Theorem Proving
- **[Diffusion-Proof: Recipe for Formal Theorem Proving Beyond Auto-Regressive Generation](https://arxiv.org/abs/2606.19315v1)** — Proposes diffusion-based methods for formal theorem proving to overcome limitations of auto-regressive token prediction in mathematical reasoning.

### Interpretability & Program Synthesis
- **[Explaining Attention with Program Synthesis](https://arxiv.org/abs/2606.19317v1)** — Uses executable program synthesis to approximate and explain the behavior of transformer attention heads in interpretable terms.

### Multi-Agent Reasoning & Decision-Making
- **[Enhancing Decision-Making with Large Language Models through Multi-Agent Fictitious Play](https://arxiv.org/abs/2606.19308v1)** — Applies game-theoretic multi-agent techniques to improve LLM-based decision-making by reasoning from multiple stakeholder perspectives.

### User Simulation & Learning
- **[Learning User Simulators with Turing Rewards](https://arxiv.org/abs/2606.19336v1)** — Proposes Turing-based reward mechanisms for training LLMs to simulate realistic human users in interactive settings, advancing agent training and system evaluation.
