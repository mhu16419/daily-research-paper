## Papers by Theme

### Agents & Reinforcement Learning
- **[Why Multi-Step Tool-Use Reinforcement Learning Collapses and How Supervisory Signals Fix It](https://arxiv.org/abs/2606.26027v1)** — Identifies catastrophic collapse in RL-based tool-use for LLMs and proposes supervisory signal interventions to restore performance.
- **[Neglected Free Lunch from Post-training: Progress Advantage for LLM Agents](https://arxiv.org/abs/2606.26080v1)** — Shows that process reward models from standard RL post-training can provide step-level supervision for long-horizon agentic tasks without additional annotation.

### Code Generation & Diversity
- **[On-Policy Self-Distillation with Sampled Demonstrations Reduces Output Diversity](https://arxiv.org/abs/2606.26091v1)** — Demonstrates that on-policy self-distillation improves pass@1 but decreases rollout diversity and flattens pass@k curves due to compounding bias in code generation.

### Program Analysis & Reverse Engineering
- **[RevengeBench: Reverse Engineering Code-Space Policies from Behavioral Experiments](https://arxiv.org/abs/2606.26094v1)** — Proposes methods to reconstruct executable decision programs from agent behavioral traces, framing code recovery as an inverse problem.
