# Daily Papers — 2026-06-27

> Topics: LLM for Code, LLM for Software Engineering

## Daily Highlight

Today's collection reveals a strong focus on **LLM-based agents and multi-agent systems for software engineering**, with papers addressing test-time scaling, design patterns, and automated evaluation of LLM outputs. Beyond agents, research highlights improvements in LLM reasoning without ground-truth supervision and the intersection of language models with specialized domains like security and hardware/software co-design.

## Papers by Theme

### LLM-Based Agents & Multi-Agent Systems
- **[Trae Agent: An LLM-based Agent for Software Engineering with Test-time Scaling](https://www.semanticscholar.org/paper/15a9b53dc8313eb0542dac00e1ea6e3dad2bbf51)** — Introduces test-time scaling techniques for LLM-based agents to improve software issue resolution through ensemble reasoning.
- **[LLM-Based Multi-Agent Systems for Software Engineering: Literature Review, Vision, and the Road Ahead](https://www.semanticscholar.org/paper/027d9a3371c13ea567bca75e675db295022a300b)** — Surveys the integration of LLMs into multi-agent systems for autonomous software engineering, highlighting cognitive planning and reasoning capabilities.
- **[From LLMs to LLM-based Agents for Software Engineering: A Survey of Current, Challenges and Future](https://www.semanticscholar.org/paper/987c5ab507b2fce644a9b73a8f8338c80cba944d)** — Comprehensive survey examining LLM agent evolution, current achievements in code generation and vulnerability detection, and open limitations.
- **[Nemotron-CORTEXA: Enhancing LLM Agents for Software Engineering Tasks via Improved Localization and Solution Diversity](https://www.semanticscholar.org/paper/34d96fe3d9c45a4a3f8e8d96aa3e54a35df935f)** — Enhances LLM agents for SE tasks by improving localization and increasing solution diversity in problem-solving.
- **[Designing LLM-based Multi-Agent Systems for Software Engineering Tasks: Quality Attributes, Design Patterns and Rationale](https://www.semanticscholar.org/paper/4fe99230d170b6c17570ec1cb4b10c3e99aef56e)** — Establishes design patterns and quality attributes for building LLM-based multi-agent systems in software engineering contexts.
- **[CHIA: An open-source framework for principled, agentic AI-driven hardware/software co-design research](https://arxiv.org/abs/2606.27350v1)** — Provides an open framework for deploying agentic AI systems at scale in hardware/software co-design across computer architecture and compilers.

### Evaluation & Validation of LLMs
- **[Automated Validation of LLM-based Evaluators for Software Engineering Artifacts](https://www.semanticscholar.org/paper/01b881d5a3471d4c0270b07c77604b9ebdf5fc95)** — Introduces REFINE, a method for automatically validating LLM-based evaluators without relying on expensive human assessments.
- **[When are likely answers right? On Sequence Probability and Correctness in LLMs](https://arxiv.org/abs/2606.27359v1)** — Investigates the fundamental alignment between sequence probability under language models and actual answer correctness.

### Security & Code Analysis
- **[Large Language Model (LLM) for Software Security: Code Analysis, Malware Analysis, Reverse Engineering](https://www.semanticscholar.org/paper/605a8b0d425a7cf6a0e97eafa681e656081d958e)** — Surveys LLM applications in malware detection, code analysis, and reverse engineering for cybersecurity tasks.
- **[LLM-Based Examination of Eligibility Criteria from Securities Prospectuses at the German Central Bank](https://arxiv.org/abs/2606.27316v1)** — Applies LLMs to extract and verify complex eligibility criteria from semi-structured, multilingual financial documents.

### LLM Training & Optimization
- **[Reinforcement Learning without Ground-Truth Solutions can Improve LLMs](https://arxiv.org/abs/2606.27369v1)** — Proposes RiVER, a ranking-induced framework enabling RL-based LLM training on optimization tasks without requiring ground-truth answers.

### Multimodal & Vision-Language Models
- **[Ask, Solve, Generate: Self-Evolving Unified Multimodal Understanding and Generation via Self-Consistency Rewards](https://arxiv.org/abs/2606.27376v1)** — Introduces a self-evolving framework enabling unified multimodal models to improve both visual understanding and generation without human annotations.
- **[Paying More Attention to Visual Tokens in Self-Evolving Large Multimodal Models](https://arxiv.org/abs/2606.27373v1)** — Addresses visual grounding in self-evolving multimodal models by ensuring attention mechanisms attend to visual content rather than relying on language priors.
- **[ViQ: Text-Aligned Visual Quantized Representations at Any Resolution](https://arxiv.org/abs/2606.27313v1)** — Proposes unified discrete representations for text and vision to enable simpler multimodal modeling while preserving low-level and semantic information.

### Vision & Perception
- **[DnA: Denoising Attention for Visual Tasks](https://arxiv.org/abs/2606.27372v1)** — Introduces Denoising Attention to reduce noise in multihead attention patterns for improved performance in visual perception tasks.
- **[Don't Settle at the Mode! Mitigating Diversity Collapse in Pretrained Flow Models via Feature Self-Guidance](https://arxiv.org/abs/2606.27371v1)** — Addresses diversity collapse in flow-based image generation through feature self-guidance without external reward models.

### Natural Language Processing & Information Extraction
- **[Mapping Political-Elite Networks in Europe with a Multilingual Joint Entity-Relation Extraction Pipeline](https://arxiv.org/abs/2606.27347v1)** — Develops a multilingual NER/RE pipeline for extracting complex political elite networks from text at scale.
- **[Beyond Surface Forms: A Comprehensive, Mechanism-Oriented Taxonomy of Indirect Linguistic Encoding for LLM-Based Coded Language Detection](https://arxiv.org/abs/2606.27314v1)** — Proposes a mechanism-oriented taxonomy of indirect linguistic expressions (algospeak, euphemisms) for detecting coded language in LLMs.

### Robotics & Embodied AI
- **[Scalable Behavior Cloning with Open Data, Training, and Evaluation](https://arxiv.org/abs/2606.27375v1)** — Releases ABC, the largest open-source teleoperation dataset with 130K episodes, enabling scalable behavior cloning research.
- **[World Action Models Enable Continual Imitation Learning with Recurrent Generative Replays](https://arxiv.org/abs/2606.27374v1)** — Proposes REGEN, enabling continual imitation learning by using generative world models to replay past tasks without storing original demonstrations.

### Healthcare & Applications
- **[Language-Based Digital Twins for Elderly Cognitive Assistance](https://arxiv.org/abs/2606.27334v1)** — Develops language-based digital twins leveraging LLMs for early detection of mild cognitive impairment in elderly populations.
